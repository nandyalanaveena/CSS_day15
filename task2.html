Transform Properties in CSS

  • In CSS, the transform property is used to apply various transformations to an element, such as rotating, scaling, skewing, or translating (moving) it.
  • It allows you to manipulate the appearance and position of elements on a web page without affecting the document's normal flow.

Property No:1:- Translate

It is used to moves an element along the horizontal and vertical axes. It takes two values, translateX () for horizontal movement and translateY () for vertical movement. 

        It has 3 attributes. They are:
                  1. translateX
                  2. translateY
                  3. translate

1. translateX: (along x-axis)

2. translateY: (along y-axis)



3. translate: (both x and y axis)

Property No:2:- Rotate

It is used to rotates an element clockwise or counterclockwise. It takes an angle value as the parameter. For example, transform rotate(45deg); rotates the element 45 degrees clockwise. 

        It has 3 attributes. They are:
                  1. rotateX
                  2. rotateY
                  3. rotate

1. rotateX: (along x-axis)

3. rotateY: (along y-axis)

3. rotate: (both x and y axis)

Property No:3:- Scale

It is used to Scales an element up or down in size. It takes one or two values to specify the scaling factor. 

        It has 3 attributes. They are:
                  1. scaleX
                  2. scaleY
                  3. scale

1. scaleX: (along x-axis)

3. scaleY: (along y-axis)

1. scale: (both x and y axis)

Property No:4:- Skew

It is used to Skews an element along the horizontal or vertical axes. It takes two angle values as parameters, skewX () for horizontal skewing and skewY () for vertical skewing. 

        It has 3 attributes. They are:
                  1. skewX
                  2. skewY
                  3. skew

1. skewX: (along x-axis)

3. skewY: (along y-axis)


3. skew: (both x and y axis)
